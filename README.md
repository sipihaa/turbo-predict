# turbo-predict

Проект по прогнозированию работы газотурбинных установок, выполненный в рамках 5-го семестра.

***

## Описание проекта

Этот репозиторий содержит код и материалы, связанные с задачами анализа и прогнозирования работы газотурбинных установок. В проекте реализованы методы нарезки данных, обучения моделей и оценки их качества. Данные версионируются с помощью DVC и хранятся в S3-совместимом хранилище.

***

## Установка виртуального окружения

Для установки необходимых зависимостей используйте команду:

```bash
poetry install --no-root
```

Это создаст необходимое окружение с библиотеками, указанными в файле `pyproject.toml`.

## Настройка DVC и получение данных

1. Попросите у меня файл с ключами доступа. Для доступа к S3-хранилищу вам понадобится файл .dvc/config.local, который содержит секретные ключи.

2. Поместите полученный файл .dvc/config.local в папку .dvc/ в корне вашего проекта.

3. Загрузите данные. После того как файл с ключами будет на месте, выполните команду для загрузки текущей версии данных из S3-хранилища:

```bash
dvc pull
```

Теперь все необходимые данные находятся в папке data/, и вы готовы к работе.

***

## Работа с данными

### Нарезка данных

Процесс подготовки данных включает создание обучающих и тестовых выборок с помощью следующей команды:

1. Создание обучающих выборок
```bash
python -m scripts.sample_creator_unit_auto -w 50 -s 1 --test 0 --sampling 10
```

2. Создание тестовых выборок
```bash
python -m scripts.sample_creator_unit_auto -w 50 -s 1 --test 1 --sampling 10
```

Объяснение параметров:

- `-w 50` — размер окна для нарезки данных.
- `-s 1` — шаг сдвига окна.
- `--test 0` или `1` — режим работы (обучение или тестирование).
- `--sampling 10` — частота выборки.
